GLX Explorer (GLX.info)

The GLX Explorer and Analytics Platform is a web tool that allows you to search and view information about blocks, addresses, tokens, and transactions on the GLX Chain.

In the future GLX will build machine readable stats and GLX Chain info that can be accessed directly through REST and Websockets APIs.

GLX Token (GLXT)

The GLX Token is a new series of digital tokens built on the Komodo Platform. They function as the exclusive payment protocol for the GLX.com ecosystem to incentivize key actions and needs within the platform.
